Thoughts on Progressive Enhancement

Progressive enchancement is a developmental approach that buils off a reliable but simple foundation. You start off using basic HTML then layer in CSS for styling and JavaScript for a more reactive website. The enhancements are optional and if a browser does not support certain features, the users still get a working website. Progressive enchancement is important as it allows for performance optimization. This can lead to less lag if people had poor connections whichs allows for the website to run and load faster.
